# SpeebrunConsistencyTracker

A mod designed to help speedrunners measure consistency during practice. This mod tracks segment times and displays real-time statistics so you can focus on repeatability and consistency, not just PBs

## How to use

1. Set a target time for the current segment
2. Practice the segment
3. Export statistics to clipboard (csv format)

## Tracked Statistics

- History of completed segment runs
- Average
- Median
- Best
- Worst
- Standard Deviation
- 90th percentile
- Number of completed runs
- Completion Rate
- Success Rate

The in-game overlay displays only the average, median, completed runs count and success rate
